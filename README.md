# **Simple Music Player using Javascript**
I have just made a simple music player using Javascript. By using basic knowledge, this is one of my first Javascript project to practice and improve my front-end developement skill.

---

## üöÄ Features
This music player contains these following feature:
- [1] **Render songs automatically** from a `songs` object.
- [2] **Scroll top effect**: When you scroll down, the CD thumbnail will vanish gradually.
- [3] **Play / Pause / Seek** (Control audio progress).
- [4] **CD rotation**: the CD will rotate automatically when the song is played and will reset when user changes song.
- [5] **Next / Previous button**: allows user to navigate through the playlist. 
- [6] **Random songs**: user can use random button to randomly move to a new song and enjoy.
- [7] **Next / Repeat when end**: when the song end, the player will move to the next one in the playlist, or move randomly if the random button is on, or repeat the current song if the user want.
- [8] **Active song**: The current song will be noticeable, so we can easily recognize what song is playing in the playlist.
- [9] **Scroll active song**: This function forces the current song in the playlist to appear in the user viewport by scrolling the window.
- [10] **Play song when click**: User can choose the song they want not just by control buttons but also by clicking directly on the playlist.
- [11] **Config**: To save the users' latest settings.
